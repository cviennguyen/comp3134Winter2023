1.What is the definition of server port?
  Web server ports are the logical endpoints of a 
  network connection that is used to exchange information between 
  a web server and a web client. 
2.What is server port range?
  Port numbers range from 0 to 65536
3.How many bits does number above represent?
  16 bit
